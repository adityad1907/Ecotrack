# Ecotrack

🌿 EcoTrack is an interactive environmental action tracking platform that motivates individuals to contribute to sustainability through gamification. Users can register, upload photos of planted saplings, and earn points through a built-in pixel-based plant verification system. Verified submissions increase their score, unlock achievement badges, and improve their rank on the global leaderboard.

The platform also includes an educational section to spread awareness about environmental impact and a marketplace for eco-friendly products. Built using HTML, CSS, JavaScript, and React (via CDN), EcoTrack runs directly in the browser without requiring backend setup, making it lightweight and easy to deploy.

EcoTrack combines technology, community engagement, and environmental responsibility to encourage real-world green actions in a fun and competitive way. 🌍
